installation

 

Before you install, make sure you have everything what you need. The AMS will operate on any IBM compatible PC, with a CD-ROM drive, and 32 MB RAM, and after installation, the software will take 30 MB of hard disk space.

 
Content of the package

 

This package will be available in a CD, which will contain the installation program and system files.

 

Install AMS

 

  •  INSERT the CD in the CD-ROM drive.

  • Go to Windows Explorer and SELECT the CD-ROM drive (D:\ or E:\).

  • Double click on Setup.exe and follow the AMS Set-up guidelines.

  • After installation, Click starts menu and selects programs and then selects AMS, containing executable file AMS.

  • Click AMS application and then the Login dialog box appears on the screen.
